### RPG Gora 입니다

### 설명
```
멀티플레이 가능한 네트워크 통신 정도만 구현됨

현재 깃 저장소는 크게 backend, front, server, infla, client, main(서브모듈 합치는 프로젝트 실행시 필수) 이렇게 나누었습니다.
통신 프로토콜(protobuf), 네트워크 라이브러리(netty), backend/server(Spring boot)를 사용했습니다.
```
